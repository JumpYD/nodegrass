nodegrass

nodegrass is a tool to process client request for Node.js

Download this project as a .zip file Download this project as a tar.gz file

What's nodegrass

nodegrass is a tool to process client request like jquery's ajax for Node.js

Install

$ npm install nodegrass

Useage

var nodegrass = require('nodegrass');
nodegrass.get("https://github.com",function(data,status,headers){
    console.log(status);
    console.log(headers);
    console.log(data);
},'utf8').on('error', function(e) {
    console.log("Got error: " + e.message);
});
